# InvestTrack Tutorial

**Target audience:** CS students who know basic JavaScript and Git but are new to React, Vite, Vitest, and GitHub Actions.

**Learning objectives**
- Set up a React + Vite project and run it locally.
- Add a unit test (Vitest) and make it pass.
- Automate tests with GitHub Actions on every push.
- (Optional) Deploy the site automatically after tests pass.
